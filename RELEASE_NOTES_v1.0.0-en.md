# Yakit English UI v1.0.0

This is the first release of the unofficial English-language fork/localization of Yakit.

Original project: [https://github.com/yaklang/yakit](https://github.com/yaklang/yakit)

This release focuses on translating the Yakit user interface and documentation from Chinese to English while preserving the original project behavior, structure, license, attribution, and security/legal disclaimers.

This is an unofficial English-language fork/localization of Yakit.

No intentional functional changes were made unless explicitly listed.

License remains AGPL-3.0.

## What changed

* Translated Chinese UI text to English.
* Translated documentation to English.
* Added/updated attribution notices.
* Verified license and upstream attribution.
* No intentional functional changes.

## Known limitations

* Some upstream terms, product names, protocol names, and technical identifiers are intentionally preserved.
* Future upstream changes may require re-translation.

## Desktop artifacts

When desktop release assets are built from this repository, the intended public artifacts are:

* Windows: `.exe` installer
* macOS: `.dmg` for `x64` and `arm64`
* Linux: `.AppImage` for `amd64` and `arm64`
